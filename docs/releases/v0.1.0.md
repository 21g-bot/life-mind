# LIFE-Mind v0.1.0（Windows 预览版）

这是 LIFE-Mind 开源框架的第一个可下载预览版，也是一个无需安装 Python 的 Windows x64
便携桌宠。它包含心智内核、像素桌宠参考宿主、状态驱动动作、黑箱个人房间、本地记忆治理，
以及 Ollama、OpenAI 兼容接口和 Anthropic 接口适配器。

## 下载与启动

1. 下载 `LIFE-Mind-v0.1.0-windows-x64.zip`；
2. 可选：用同名 `.sha256` 文件核对下载完整性；
3. 完整解压 ZIP，不要直接在压缩包里运行；
4. 双击 `LIFE-Mind.exe`。

首次运行会在 `%LOCALAPPDATA%\LIFE-Mind\` 生成中立的“小芽（演示）”动作库和本地运行数据。
发行包不包含维护者的私人角色素材、对话、记忆、日记、密钥或用户档案。

如需替换角色，可把自己的动作库放在 `LIFE-Mind.exe` 同级的 `character\` 目录，或设置
`LIFE_MIND_ASSET`。动作库格式见仓库的角色教程。

## 重要说明

- 这是预览版，不宣称已经通过正式 MVP 所要求的 8 小时桌面稳定性和真实体验者盲测；
- 当前便携 EXE 没有商业代码签名证书，Windows SmartScreen 可能提示“未知发布者”；
- 请只从 `21g-bot/life-mind` 的 GitHub Releases 下载，并用 SHA256 校验文件核对；
- AI 是可选项。未配置模型时离线状态机、记忆和动作仍然工作；
- 云端模型只在用户明确许可后发送必要上下文，API 密钥保存在 Windows 凭据管理器。

已自动验证：公开边界扫描、完整单元测试、PyInstaller Windows 构建、ZIP 内容检查、SHA256
一致性，以及在隔离用户目录执行冻结版 `LIFE-Mind.exe --release-check`。
