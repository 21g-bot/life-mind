# LIFE-Mind v0.1.1（Windows 预览版）

这是首个维护更新，重点修复配置容错、AI 接口传输安全、凭据隔离和慢模型请求期间的桌面
响应问题。已有的本地记忆、房间数据和角色动作库不会因为替换程序目录而被清空。

## 本次改进

- 桌宠配置文件即使被写成数组等错误 JSON 结构，也会安全恢复默认值，不再导致启动崩溃；
- 严格解析窗口坐标、缩放和布尔设置，避免 `"false"` 被错误当成开启；
- 远程与局域网 AI 接口必须使用 HTTPS，防止对话和 API 密钥以明文经过网络；
- 凭据绑定保留大小写敏感的 URL 路径，避免多租户网关复用错误密钥；
- 慢模型生成期间不再阻塞桌面自主刷新，重复对话会被拦截，退出窗口也会立即响应；
- Windows 产物工作流已迁移到 Node.js 24 版本的官方 Artifact Actions。

本机回环地址 `localhost`、`127.0.0.1` 和 `::1` 仍可使用 HTTP，因此默认 Ollama、LM Studio
和其他本机兼容服务不受影响。部署在局域网其他设备上的模型需要先配置 HTTPS 反向代理。

## 下载与升级

1. 下载 `LIFE-Mind-v0.1.1-windows-x64.zip` 和同名 `.sha256` 文件；
2. 核对 SHA256 后完整解压，不要直接在压缩包预览窗口运行；
3. 退出旧版本，再双击新目录中的 `LIFE-Mind.exe`。

运行数据默认保存在 `%LOCALAPPDATA%\LIFE-Mind\`，不位于便携程序目录中。替换版本前仍建议
保留自己的角色动作库原件；公开发行包只包含中立的“小芽（演示）”，不会包含维护者的私人
角色素材、对话、记忆、日记、密钥或用户档案。

## 重要说明

- 这是预览版，尚未宣称通过正式 MVP 所要求的 8 小时桌面稳定性和真实体验者盲测；
- 当前 EXE 没有商业代码签名证书，Windows SmartScreen 可能显示“未知发布者”；
- 请只从 `21g-bot/life-mind` 的 GitHub Releases 下载，并核对随包提供的 SHA256；
- AI 是可选项；未配置模型时，离线心智、记忆、动作和黑箱个人房间仍然工作。

已自动验证：124 项单元测试、三个公开协议样例、公开边界与文档链接检查、像素动作库严格
QA、PyInstaller Windows x64 构建、ZIP/PE/SHA256 检查，以及在隔离用户目录实际执行冻结版
`LIFE-Mind.exe --release-check`。
