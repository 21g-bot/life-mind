# LIFE-Mind v0.1.2（Windows 预览版）

这是一次长期运行性能与 AI 输入边界维护更新。已有本地记忆、房间数据和角色动作库不会因为
替换程序目录而被清空。

## 本次改进

- 心智事件改为增量追加，不再在每次互动时回放全部历史；500 条基准由约 58.3 秒降至约
  1.39 秒；
- 增量计算或 SQLite 写入失败时自动回滚到最后一次已提交状态，保持内存与数据库一致；
- 重复事件 ID 使用常数时间查找并保持幂等，增量状态仍与完整确定性回放完全一致；
- 单条用户消息限制为 4000 个字符，超限内容不会写入数据库或发送给模型；
- 模型 HTTP 响应限制为 4 MiB，异常网关无法用超大 JSON 持续占用内存；
- 严格拒绝字段类型错误、布尔伪装数值或非有限数值等不符合结构化契约的模型输出；
- 检测到覆盖系统规则、窃取密钥或绕过授权等提示注入形态时，该轮文本不会成为长期记忆；
- 模型请求的 `User-Agent` 携带真实 LIFE-Mind 版本，方便兼容服务诊断。

DeepSeek、智谱 GLM、Gemini、Kimi 和 Anthropic Claude 的预设地址与模型名称已对照各厂商
当前官方文档复核。本次没有改变已有 AI 配置或凭据绑定，升级后无需重新填写密钥。

## 下载与升级

1. 下载 `LIFE-Mind-v0.1.2-windows-x64.zip` 和同名 `.sha256` 文件；
2. 核对 SHA256 后完整解压，不要直接在压缩包预览窗口运行；
3. 退出旧版本，再双击新目录中的 `LIFE-Mind.exe`。

运行数据默认保存在 `%LOCALAPPDATA%\LIFE-Mind\`，不位于便携程序目录中。公开发行包只包含
中立的“小芽（演示）”，不会包含维护者的私人角色素材、对话、记忆、日记、密钥或用户档案。

## 重要说明

- 这是预览版，尚未宣称通过正式 MVP 所要求的 8 小时桌面稳定性和真实体验者盲测；
- 当前 EXE 没有商业代码签名证书，Windows SmartScreen 可能显示“未知发布者”；
- 请只从 `21g-bot/life-mind` 的 GitHub Releases 下载，并核对随包提供的 SHA256；
- AI 是可选项；未配置模型时，离线心智、记忆、动作和黑箱个人房间仍然工作。

已自动验证：131 项单元测试、三个公开协议样例、公开边界与文档链接检查、100 次确定性
回放、像素动作库严格 QA、PyInstaller Windows x64 构建、ZIP/PE/SHA256 检查，以及在隔离
用户目录实际执行冻结版 `LIFE-Mind.exe --release-check`。
